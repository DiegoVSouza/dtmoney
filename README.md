## 👨‍💻 Rodando na sua máquina

**1:** Clone o repositório.

```
git clone https://github.com/DiegoVSouza/dtmoney.git

```

**2:** Acesse o diretório do projeto.

```
cd dtmoney

```

**3:** Instale as depedências
```
    yarn

```
**4:** Inicie a aplicação.
```
    yanr start
   
```
